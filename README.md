# 30 Days SOC Challenge

Welcome to the **30 Days SOC Challenge** repository! This project provides a structured, hands-on set of labs and exercises covering log analysis, network monitoring, incident response, threat intelligence, malware analysis, and endpoint detection & response (EDR).

---


## Overview

This repository contains 30 practical labs intended to build skills relevant to SOC roles. Each day focuses on a specific topic with instructions, commands, screenshots, and observations to help you practice detection and response techniques.

## Daily Curriculum

### Day 1–5: Log Analysis Basics

1. **Day 1:** Introduction to Log Analysis
2. **Day 2:** Log Analysis Basics: Windows Security Logs
3. **Day 3:** Log Analysis Basics: Windows PowerShell Logs
4. **Day 4:** Log Analysis Basics – Network-Based Attack Detection Using UFW
5. **Day 5:** Log Analysis Basics – Linux Auth Log

### Day 6–10: Wireshark / Network Analysis

6. **Day 6:** Introduction to Wireshark – Packet Analysis for SOC Analysts
7. **Day 7:** Wireshark Basics – ICMP Protocol Analysis
8. **Day 8:** Wireshark Basics – TCP Protocol Analysis
9. **Day 9:** Wireshark Basics – HTTP Protocol Analysis
10. **Day 10:** Wireshark Basics – TLS Protocol Analysis

### Day 11–15: Incident Response

11. **Day 11:** Introduction to Incident Response
12. **Day 12:** Incident Response Basics – Linux Suspicious Bash Script Execution
13. **Day 13:** Detecting and Removing Malicious Cron Jobs – Linux IR Lab
14. **Day 14:** Detecting Suspicious PowerShell Activity – Windows IR Lab
15. **Day 15:** Incident Response Basics – Suspicious Network Connection

### Day 16–20: Splunk Basics

16. **Day 16:** Install and Configure Splunk
17. **Day 17:** Splunk Basics – DNS Log Analysis
18. **Day 18:** Splunk Basics – SSH Log Analysis
19. **Day 19:** Splunk Basics – HTTP Log Analysis
20. **Day 20:** Splunk Basics – Zeek Connection Log Analysis and Investigating Unauthorized Access

### Day 21–24: Phishing, Threat Intel & Malware

21. **Day 21:** Introduction to Phishing Analysis
22. **Day 22:** Phishing Analysis – Investigating a Phishing Email
23. **Day 23:** Threat Intelligence Basics
24. **Day 24:** Introduction to Malware Analysis

### Day 25–30: Digital Forensics & EDR

25. **Day 25:** Introduction to Digital Forensics
26. **Day 26:** Setting up Wazuh
27. **Day 27:** EDR Basics: File Integrity Monitoring (FIM) using Wazuh
28. **Day 28:** EDR Basics: Detecting SSH Brute Force Attack
29. **Day 29:** EDR Basics: Detecting Suspicious Network Traffic using Suricata
30. **Day 30:** EDR Basics: Vulnerability Detection using Wazuh

## Tools & Platforms

* Kali Linux (attacker simulation)
* Ubuntu Desktop (Linux labs)
* Windows 10/11 (Windows labs)
* Hydra (password testing)
* Wireshark (packet capture and analysis)
* Splunk (SIEM)
* Wazuh (EDR and endpoint monitoring)
* Suricata (network IDS)
* Shell utilities: 'grep', 'awk', 'tail', etc.
